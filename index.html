Section 1: CLI (5%)

Question 1: What is CLI?
Answer: CLI is stands for "Command Line Interface" - a command line program that get text input to execute operating system. At the beginning of computer era, there is no User Interface like macOS or Windows 11, only way to work and interact and "command" to computer is computer terminals.
Question 2: What is the advantage/disadvantage of CLI?
Answer:
Pros:
  • Command and interact with operating system (OS) faster, shortcut typing command without many mouse clicks
  • Save memory and CPU consumption, runnable on any screen with low resolution (limiting as the beginning), due to simple, lightweight User Interface, just solid black background and white text
Cons:
  • Need a high learning curve before get familiar with it
  • Need to learn by hear all - a lot command text, if not typing correctly it will fails
Question 3: Which do you prefer, CLI or GUI (Graphical User Interface)? Why?
Answer: I prefer CLI in the most case, because it faster in almost operation when coding, I also try to using more shortcut with other apps besides Terminal, but some operation especially when working with Front-End I must user GUI.
Question 4: Which command to check your current directory?
Answer: In linux Terminal I user pwd that stands for Print Working Directory In windows Terminal we can use cd as a tip for equivalent result
Question 5: Which command to create a new folder/directory?
Answer: We can use mkdir to create new folder/directory.
Question 6: Which command to create a new file?
Answer: We can use touch index.html to create new file name index.html. For Windows Terminal it's little tricky by using type nul > index.html to create empty file
Question 7: You are in Desktop folder, which command to make a new directory name coderschool and navigate inside it?
Answer: We can use mkdir coderschool && cd coderschool at same line to quick navigate
Question 8: Which command to delete a file?
Answer: We can use rm index.html to delete index.html file.
Question 9: Which command to delete a not-empty folder?
Answer: We can use rm -rf index.html to delete not-empty folder. The –recursive or -r option with the rm command will delete the files and sub-folders within it recursively. the –force or -f option to avoid getting prompted each time deleting each sub-folder and every file.
Question 10: What does this command do?
Answer: echo "# test" >> README.md will create a new README.md file if this file not exist and add # test to it's content or if there is a README.md already exist it will add # test to it's content bottom line.

Answer: This tag specifies the character encoding for the HTML document. Mostly web developers to use the UTF-8 character set, due to it covers almost all of the characters and symbols.
Question 17: Inside head tag, what does second meta tag mean?
Answer: This tag let we take control over the viewport. This will affect the page's dimensions and scaling.
  • The width=device-width part sets the width of the page to follow the screen-width of the variable width device.
  • The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.

    Section 4: CSS (10%)

    Question 26: Explain box model elements.
    Answer: All the elements in an HTML document as rectangular boxes with their own dimensions. These boxes contain a content area and optional surrounding margin, border, and padding areas.
    Question 27: The value in inch of 1 px?
    Answer: It's depend on the dpi (pixel density or dots per inch). Assuming the pixel density is 100 dpi, there are 100 pixels per inch. Then 1 pixel = (1 / 100) inches.So for this case value of 1px in inch is 0.01 inches.
    Question 28: What is vw/vh?
    Answer: vh stands for viewport height and vw is for viewport width. So, setting an element to a width value of 50vw means that the element will have a width that’s 50% of the viewport size even when the viewport is resized.
    Question 29: List all values of position property? Which one is default value?
    Answer: There are five different position values:
    • static
    • relative
    • fixed
    • absolute
    • sticky
    The default value is position:static
    Question 30: List all values of flex-direction property? Which one is default value?

    • column-reverse
